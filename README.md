# Customer_Behaviour_Analysis
Customer Shopping Behaviour Analysis
An end-to-end data analytics project covering data cleaning, SQL analysis, and business intelligence dashboard — built using Python, MySQL, and Power BI.

📌 Project Overview
This project analyses customer shopping behaviour across 3,900 customers to uncover revenue patterns, subscription trends, and product performance insights. The workflow covers the full analytics pipeline — from raw data cleaning in Python, to relational querying in MySQL, to an interactive Power BI dashboard.

🔍 Business Questions Answered (SQL)

Revenue by gender — What is the total revenue generated by male vs. female customers?
Discount effectiveness — Which customers used a discount but still spent above average?
Top-rated products — Which 5 products have the highest average review rating?
Shipping comparison — How does average spend differ between Standard and Express shipping?
Subscription value — Do subscribed customers spend more? Revenue and count comparison.
Discount adoption — Which 5 products have the highest discount usage rate?
Customer segmentation — Classify customers as New, Returning, or Loyal based on purchase history.
Repeat buyer loyalty — Are repeat buyers (5+ purchases) more likely to subscribe?
Age group revenue — Which age group contributes the most to total revenue?


🐍 Python Notebook Highlights

Handled missing values using median imputation per category for Review Rating
Standardised column names (snake_case)
Engineered new features:

age_group — segmented into Young Adult, Adult, Middle Aged, Senior
purchase_frequency_days — converted text frequency to numeric days


Removed redundant column (promo_code_used was 100% identical to discount_applied)
Exported cleaned data directly to MySQL using SQLAlchemy


📈 Key Findings

Middle-aged customers generate the highest revenue ($89K), followed by Adults ($66K)
Clothing is the top revenue category ($104K) and most purchased category (1,737 items)
Only 27% of customers are subscribed — a significant growth opportunity
Subscribed customers tend to have higher average spend and total revenue contribution
73% of customers are non-subscribers, suggesting a large untapped retention segment


🗄️ Dataset

Source: Kaggle — Customer Shopping Behaviour Dataset
Rows: 3,900 customers
Columns: 18 features including Age, Gender, Category, Purchase Amount, Subscription Status, Shipping Type, Review Rating, and Previous Purchases
